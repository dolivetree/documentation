# Playground

Here you can freely try the Decipad:

```deci live
Salary = 2000€
Rent = 500€
FoodAndStuff = 400€
Savings = Salary - Rent - FoodAndStuff
==> 1100 €
```

Maybe calculate how much you weight in bananas?

```deci live
You = 50kg
Banana = 118g

round(You in Banana)
==> 424 Bananas
```
